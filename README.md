# RedScan
# Description:
Redscan is a fast port scanner, to quickly determine what ports are open on a machine, it has a friendly CLI interface, where you type the target into the promt
# Requirements:
Preferrably linux, it's fitting for kali or parrot, but it would work the same on basically any distro
# Usage: 
On windows, open CMD and cd into the folder you installed it to, run "python Redscan.py" on linux, "python3 Redscan.py"
